## Файлы для CommentTree — древовидные комментарии с навигацией и поиском  

### 📋 Описание проекта  

Комментарии — неотъемлемая часть любого интерактивного сервиса: форума, соцсети, блога, маркетплейса.  
Почти всегда они имеют древовидную структуру: пользователи отвечают друг другу, и цепочка обсуждений  
уходит вглубь. Такое поведение сложно реализовать просто в лоб — приходится учитывать вложенность,  
удаление цепочек, поиск и т.д.

В этом проекте реализован сервис для работы с древовидными комментариями, поддерживающий  
неограниченную вложенность и дополнительные функции вроде поиска, сортировки и постраничного вывода.  

### 🖥️ Возможности  

Встроенные HTTP-методы:  

  – **POST /comments** — создание комментария (с указанием родительского);  
  – **GET /comments?parent={id}** — получение комментария и всех вложенных;  
  – **DELETE /comments/{id}** — удаление комментария и всех вложенных под ним.  

Поддержка:  
- постраничную навигацию и сортировку при выводе комментариев;  
- полнотекстовый поиск по комментариям;  
- простая веб-страница (HTML + JS) для  
    + создния новых комментариев (и ответов),  
    + удаление комментариев,  
    + поиск комментариев по ключевым словам.

### 🚀 Быстрый старт  

**Требования:**  

- Docker  
- Свободные порты: 8081 (веб-интерфейс), 5432 (PostgreSQL)  

**Docker-образы**  

- postgres:16  

**Запуск системы:**  

    cd ./ && docker compose up  

- После успешного запуска веб-интерфейс будет доступен по адресу: http://localhost:8081  

### ⚙️ Конфигурация  

Все настройки задаются через файл .env в корне проекта.

    ## переменные сервера и api
    SERVICE_HOST_NAME=0.0.0.0         # имя службы (контейнера) в докере
    SERVICE_PORT=8081                 # порт хоста, на котором работает сервис
    GIN_MODE=debug                    # переключатель режима логов (debug / release)

    ## переменные базы данных
    DB_HOST_NAME=dbPostgres           # имя службы (контейнера) в докере
    DB_PORT=5432                      # порт, на котором сидит база данных
    DB_NAME=db-postgres               # имя базы данных
    DB_PASSWORD=postgres              # пароль базы данных
    DB_USER=postgres                  # имя пользователя базы данных
